Mpment 1 - Automatisering & publicering

Syftet med uppgiften är att utforska hur NodeJS kan underlätta webbplatsutveckling genom att automatisera olika uppgifter.
Målet är att implementera kommandobaserade verktyg i NodeJS för att förenkla utvecklingsprocessen, skapa en automatiserad
utvecklingsmiljö baserad på Git och NodeJS för smidig publicering av webbplatsuppdateringar, samt automatisera transpilering och minifiering av CSS, HTML och JavaScript.
Dessutom ingår att optimera grafikfiler och utveckla en "boilerplate" för webbutveckling som kan fungera som en grund för framtida projekt.
